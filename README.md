# Covid-19-Chest-X-ray-detection
Detect covid diseases in chest x-rays using multiple Convolutional Neural Networks. 

Implemeted a research report from Nature titled "COVID‑19 detection from chest X-ray images using transfer learning".
https://www.nature.com/articles/s41598-024-61693-0#Abs1

Analyzed multiple CNN algorithms:
  - DENSE121
  - ResNet50
  - ResNet152
  - EfficientNetB0

Evaluated the results over following metrics:
  - Sensitivity/Recall
  - Specificity
  - Precision
  - Accuracy
  - F1-Score
